Opis szczegółowy

Klasa opisuje pojecie sceny jako przestrzeni, w której znajduja sie zadane obiekty. Moze zawierac wiele obiektow typu Drone. Jako pola prywatne zzwiera kontener dronow, sciezke do pliku z plaszczyzna, Lacze do programu GNUPlot i numer aktywnego drona.

void Scene::SwitchActiveDrone ( )
